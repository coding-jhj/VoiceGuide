# VoiceGuide — 강사님 설명 스크립트 (최종 구현 기준)

> 2026-04-26 기준 실제 구현 반영 버전

---

## 1. 왜 만들었나요? (문제 정의)

시각장애인이 실내를 이동할 때 가장 필요한 것은 **"지금 어떻게 움직여야 하는가"** 입니다.

기존 서비스(Google Lookout, Microsoft Seeing AI)는 "의자가 있습니다"처럼 **설명**만 합니다.  
VoiceGuide는 **"바로 앞에 의자가 있어요. 멈추세요."** 처럼 즉각적인 행동 지침을 줍니다.

---

## 2. 무엇을 만들었나요? (서비스 개요)

스마트폰 카메라로 주변을 1초마다 자동 촬영하여 장애물의 위치·거리·회피 방향을 한국어로 음성 안내합니다.

### 안내 문장 예시

| 상황 | 출력 음성 |
|------|---------|
| 1m 앞 의자 | "멈추세요! 바로 앞에 의자가 있어요. 약 1.0m." |
| 0.5m 이내 위험 | "위험! 바로 앞 의자!" |
| 바닥 가방 | "조심! 발 아래 배낭. 오른쪽으로 피해가세요." |
| 계단 감지 | "조심! 0.7m 앞에 계단이나 낙차가 있어요. 멈추세요." |
| 두 가지 장애물 | "바로 앞에 의자가 있어요. 약 1.2m. 왼쪽에 사람도 있어요." |
| 재방문 시 | "의자가 생겼어요. 오른쪽 앞에 테이블이 있어요." |
| 서버 끊김 | "서버 연결이 끊겼어요. 주의해서 이동하세요." |

## 3. 어떻게 만들었나요? (기술 구조)

### 전체 파이프라인

```
[Android 카메라] → 1초마다 JPEG 캡처
         ↓
[FastAPI /detect 서버]
  ├─ YOLO11m_indoor → 장애물 탐지 (계단 포함)
  ├─ Depth Anything V2 → depth map → 거리(m) 정제
  ├─ 깊이 맵 분석 → 계단/낙차/턱 감지 (YOLO 보완)
  ├─ 객체 추적기 (EMA) → 거리 안정화, 접근 감지
  ├─ SQLite → WiFi SSID 기반 공간 기억
  └─ 한국어 문장 생성 → 긴박도 4단계
         ↓
[Android TTS] → 음성 재생
```

## 4. 핵심 기술 4가지

### ① YOLO11m 파인튜닝 (계단 클래스 추가)

기존 YOLO11m은 COCO 80클래스로 계단이 없습니다.  
DuckDuckGo로 계단 이미지 404장을 수집, 자동 라벨링 후 파인튜닝했습니다.

- **학습 시간**: RTX 5060 GPU로 약 9분
- **계단 탐지 성능**: Precision 91.7% / Recall 100% / **mAP50 = 0.992**

### ② Depth Anything V2 — 단안 깊이 추정

카메라 이미지 한 장으로 픽셀별 깊이 맵을 생성합니다.

- YOLO bbox 중심의 깊이값 → 거리(m) 변환
- 안전 우선: 실제보다 약간 더 가깝게 추정 (조기 경고)

### ③ 깊이 맵 기반 계단/낙차 감지

**YOLO가 못 잡는 계단도 감지**합니다.  
이미지 하단 바닥 영역을 12구역으로 분할하여 깊이 변화를 분석합니다.

- 깊이 급증 → 낙차/계단 하강 → "조심! Xm 앞에 계단이나 낙차가 있어요."
- 깊이 급감 → 턱/계단 상승 → "발 앞에 턱이나 계단이 있어요."

**YOLO + Depth V2 이중 구조**로 서로 보완합니다.

### ④ WiFi SSID 기반 공간 기억

- 같은 WiFi에서는 이전 방문 정보를 SQLite에 기억
- 재방문 시 **변화만 안내**: "의자가 생겼어요", "사람이 사라졌어요"
- 경쟁 서비스에 없는 차별화 기능

## 6. 모델 학습 관련

**YOLO 파인튜닝을 직접 수행했습니다.**

- 기존: COCO 사전학습 모델 (계단 클래스 없음)
- 추가: DuckDuckGo 계단 이미지 수집 + YOLO 자동 라벨링 + 파인튜닝
- 결과: `yolo11m_indoor.pt` (계단 mAP50 = 0.992)

Depth Anything V2, Android TTS/STT는 사전학습 모델을 그대로 활용합니다.

## 8. 예상 질문 답변

**Q. 실시간인가요?**  
A. 1초마다 자동 촬영 후 분석합니다. 응답은 약 200~400ms입니다.

**Q. 인터넷 없으면?**  
A. 서버 없이 폰 단독으로 동작하는 온디바이스 모드가 있습니다. (yolo11m.onnx 탑재)

**Q. 계단은 어떻게 감지하나요?**  
A. 두 가지입니다. YOLO 파인튜닝 모델로 시각적으로 인식하고, Depth Anything V2의 깊이 변화로 낙차를 감지합니다.

**Q. 거리 정확도는?**  
A. Depth Anything V2 기반입니다. 실측 보정을 통해 정확도를 높일 수 있으며, 안전 우선으로 실제보다 약간 더 가깝게 추정합니다.

**Q. 유리문은 감지 못하나요?**  
A. 투명 장애물은 현재 한계입니다. 이는 상용 서비스도 공통적으로 갖는 한계입니다.
