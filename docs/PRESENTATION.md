# VoiceGuide 발표 가이드 (조장용)

## 한 줄 소개

> "카메라로 주변을 찍으면 장애물 위치·거리·회피 방향을 한국어로 음성 안내하는 시각장애인 보행 보조 AI 서비스"

---

## 문제 정의 (왜 만들었나)

기존 서비스(Google Lookout, Microsoft Seeing AI)는 **"의자가 있습니다"** 처럼 설명만 해요.  
VoiceGuide는 **"바로 앞에 의자가 있어요. 멈추세요."** 처럼 행동 지침을 줍니다.

시각장애인 분이 실제로 필요한 건 설명이 아니라 **"지금 어떻게 움직여야 하는가"** 입니다.

---

## 시스템 구조 (그림으로 설명)

```
[Galaxy Z Fold 4]  ──WiFi──▶  [PC 서버]
   카메라 1초 캡처              YOLO11m 탐지
   음성 안내 재생     ◀──────  Depth V2 거리
   STT 음성 명령               계단 감지
   온디바이스 ONNX              한국어 문장 생성
```

---

## 핵심 기술 4가지

### 1. YOLO11m 파인튜닝 (계단 클래스 추가)

- 기본 YOLO: 80개 COCO 클래스, 계단 없음
- DuckDuckGo로 계단 이미지 404장 수집 → 자동 라벨링
- 파인튜닝 결과: **계단 mAP50 = 0.992** (정밀도 91.7%, 재현율 100%)

**발표 포인트:** "직접 데이터 수집하고 학습해서 계단 탐지를 추가했습니다"

### 2. Depth Anything V2 깊이 추정

- 단일 이미지에서 픽셀별 깊이 맵 생성
- GPU(RTX 5060)에서 실시간 추론
- YOLO bbox + 깊이 맵 → 실제 거리(m) 계산

**발표 포인트:** "카메라 하나로 거리를 추정하는 최신 AI 모델을 적용했습니다"

### 3. 깊이 맵 기반 계단/낙차 감지

- **YOLO가 못 잡는 계단도 감지** (낙차, 턱, 좁은 통로)
- 이미지 하단 바닥 영역을 12개 구역으로 분석
- 깊이 값이 갑자기 증가 → 낙차, 감소 → 턱으로 판별

**발표 포인트:** "두 AI가 서로 보완합니다. YOLO는 물체를, Depth V2는 바닥 구조를 봅니다"

### 4. WiFi SSID 기반 공간 기억

- 같은 WiFi에서는 이전 방문 정보를 기억
- **재방문 시 변화만 안내:** "의자가 생겼어요"
- 경쟁 서비스에 없는 차별화 기능

**발표 포인트:** "방을 '기억'해서 처음 방문과 재방문을 구분합니다"

## 안내 문장 예시 (시나리오별)

| 상황 | 출력 음성 |
|------|---------|
| 1m 앞 의자 | "멈추세요! 바로 앞에 의자가 있어요. 약 1.0m." |
| 0.5m 앞 위험 | "위험! 바로 앞 의자!" |
| 바닥에 가방 | "조심! 발 아래 배낭. 오른쪽으로 피해가세요." |
| 계단 감지 | "조심! 0.7m 앞에 계단이나 낙차가 있어요. 멈추세요." |
| 좌우 장애물 | "바로 앞에 의자가 있어요. 왼쪽에 사람도 있어요." |
| 재방문 변화 | "의자가 생겼어요. 오른쪽 앞에 테이블이 있어요." |
| 서버 끊김 | "서버 연결이 끊겼어요. 주의해서 이동하세요." |

## 데모 시나리오 (실기기)

```
1. Galaxy Z Fold 4에 앱 실행
2. 서버 URL 입력: http://192.168.x.x:8000
3. "분석 시작" 버튼
4. 의자 앞에 서면 → "바로 앞에 의자가 있어요. 멈추세요." 자동 출력
5. 음성 명령 버튼 → "주변 알려줘" 말하기 → 장애물 모드 확인
6. 방 나갔다가 다시 들어오기 → "의자가 1개 있어요" (공간 기억 시연)
```

## 예상 질문 & 답변

**Q. 실시간이에요?**  
A. 1초마다 자동 캡처해서 분석합니다. 응답 시간은 약 300~500ms입니다.

**Q. 인터넷 없으면 안 되나요?**  
A. 서버 연결 없이도 폰 단독으로 동작하는 온디바이스 모드가 있습니다.

**Q. 계단은 어떻게 감지하나요?**  
A. 두 가지로 감지합니다. YOLO 파인튜닝 모델(시각적 인식)과 Depth Anything V2의 깊이 변화 분석(3D 구조 인식)이 서로 보완합니다.

**Q. 거리 정확도는?**  
A. Depth Anything V2 기반으로 추정하며, 환경에 따라 ±30% 오차가 있을 수 있습니다. 안전 우선으로 실제보다 더 가깝게 추정합니다.

**Q. 유리문은 감지 못하지 않나요?**  
A. 투명 장애물 감지는 현재 한계입니다. 추후 투명 물체 전용 데이터셋으로 개선 계획입니다.
