# VoiceGuide 발표 가이드

## 한 줄 소개

> "스마트폰 카메라로 주변을 찍으면 실내외 장애물·차량·동물의 위치와 회피 방향을 한국어로 즉시 음성 안내하는 시각장애인 보행 보조 AI 앱"

---

## 문제 정의

기존 서비스(Google Lookout, Seeing AI)는 **"의자가 있습니다"** 처럼 설명만 합니다.  
VoiceGuide는 **"바로 앞에 의자가 있어요. 멈추세요."** 처럼 즉각적인 행동 지침을 줍니다.

| 서비스 | 출력 | 한계 |
|--------|------|------|
| Google Lookout | "의자가 있습니다" | 방향·거리 없음 |
| Seeing AI | "왼쪽에 의자가 있습니다" | 행동 지침 없음 |
| **VoiceGuide** | **"바로 앞에 의자가 있어요. 멈추세요."** | — |

---

## 시스템 구조

```
[Galaxy Z Fold 4]
   카메라 1초 자동 캡처
       ↓ (서버 없이 폰 단독 동작)
   YOLO11m ONNX 온디바이스 추론
       ↓
   방향(8시~4시) + 거리(bbox 추정) + 위험도 계산
       ↓ (서버 연결 시 Depth V2로 거리 정확도 향상)
   한국어 문장 생성
       ↓
   Android TTS 즉시 음성 출력
```

## 핵심 기술

### 1. YOLO11m 파인튜닝 — 계단 클래스 추가

- COCO 기본 80클래스에 **계단(stairs)** 직접 추가 학습
- 학습 데이터: 계단 이미지 404장 (DuckDuckGo 자동 수집)
- 결과: **계단 mAP50 = 0.992** (정밀도 91.7%, 재현율 100%)

**발표 포인트:** "시각장애인에게 가장 위험한 계단을 직접 데이터 수집·학습해서 추가했습니다"

### 2. 실내외 전체 위험 요소 커버

| 환경 | 위험 요소 |
|------|---------|
| **야외** | 자동차·오토바이·버스·트럭 (위험도 3~4배), 자전거, 신호등, 개·말 |
| **실내** | 의자·테이블·소파, 바닥 가방·여행가방, 계단·낙차·턱, 사람 |

**발표 포인트:** "실내뿐 아니라 야외 차량·동물까지 모두 커버합니다. 차량은 위험도 3배로 더 강하게 경고합니다"

### 3. Depth Anything V2 깊이 추정 (서버 연동 시)

- 단일 이미지로 픽셀별 깊이 맵 생성 (NeurIPS 2024 SOTA)
- "약 1.5미터" 같은 실제 거리 안내 가능

### 4. 개인 네비게이팅 ⭐

- "여기 저장해줘 화장실" → WiFi SSID + 장소 이름 저장
- 다음에 같은 WiFi 연결 시 → "화장실이 이 곳이에요!"
- **경쟁 서비스에 없는 차별 기능**

### 5. 완전 오프라인 동작

- 서버 없이 폰 단독으로 ONNX 추론
- 인터넷 없는 환경, 지하, 비행기 안에서도 동작

## 안내 문장 예시

| 상황 | 음성 출력 |
|------|---------|
| 1.5m 앞 의자 | "바로 앞에 의자가 있어요. 약 1.5m. 멈추세요." |
| 0.4m 이내 위험 | "위험! 바로 앞 의자!" |
| 바닥 가방 | "조심! 발 아래 배낭이 있어요. 오른쪽으로 피해가세요." |
| 계단 감지 | "조심! 0.7m 앞에 계단이나 낙차가 있어요. 멈추세요." |
| **자동차 접근** | **"위험! 오른쪽에 자동차가 있어요! 약 3.0m. 즉시 왼쪽으로 피해가세요!"** |
| **개 발견** | "조심! 왼쪽 앞에 개가 있어요. 약 2m. 천천히 오른쪽으로 피해가세요." |
| 재방문 | "의자가 생겼어요. 바로 앞에 테이블이 있어요." |
| 장소 저장 | "편의점을 저장했어요." |

## 앱 설치 방법

> **플레이스토어 미출시** — 아래 방법으로 설치합니다.

### 방법 A: 직접 빌드 (권장)
```
1. Android Studio 설치
2. git clone https://github.com/coding-jhj/VoiceGuide.git
3. android/ 폴더를 Android Studio에서 열기
4. 폰 USB 연결 → USB 디버깅 ON → ▶ Run
```

### 방법 B: APK 직접 설치 (팀원용)
```
1. 조장에게 APK 파일 받기 (카카오톡/구글드라이브)
2. 폰에서 APK 파일 열기
3. 설정 → 보안 → 출처를 알 수 없는 앱 허용
4. 설치 완료
```

### 빌드 후 APK 위치
```
android/app/build/outputs/apk/debug/app-debug.apk
```

## 데모 시나리오 (실기기)

```
1. APK 설치된 Galaxy Z Fold 4 꺼내기
2. 앱 실행 → "분석 시작" 버튼 (서버 URL 입력 불필요)
3. 실내: 의자 앞에 서기 → "바로 앞에 의자가 있어요. 멈추세요." 자동 출력
4. 야외: 자전거/오토바이 방향으로 카메라 → "위험! 자전거가..." 강화 경고
5. 음성 명령 버튼 → "주변 알려줘" → 장애물 모드 확인
6. 음성 명령 → "여기 저장해줘 강의실" → "강의실을 저장했어요."
7. 잠시 후 같은 장소 → "저장된 곳 알려줘" → "저장된 장소는 강의실이에요."
```

## 예상 질문 & 답변

**Q. 실시간인가요?**  
A. 1초마다 자동 캡처·분석합니다. 온디바이스 추론 응답은 약 50~100ms입니다.

**Q. 인터넷이 없어도 되나요?**  
A. 네. 폰에 ONNX 모델이 내장되어 있어 완전 오프라인 동작합니다.

**Q. 야외에서 빠른 차량도 감지하나요?**  
A. 가까운 차량은 bbox 크기가 커서 잘 감지됩니다. 빠르게 접근하는 차량의 경우 confidence 임계값을 0.40으로 낮춰 일찍 감지하도록 설정했습니다.

**Q. 신호등 색깔도 알려주나요?**  
A. 현재는 신호등 위치만 안내합니다. 색깔 분류는 향후 추가 가능한 기능입니다.

**Q. 계단은 어떻게 감지하나요?**  
A. YOLO 파인튜닝(시각 인식) + Depth V2 깊이 변화 분석(바닥 구조 인식) 두 가지로 감지합니다.

**Q. 거리 정확도는?**  
A. Depth V2 없이는 ±50%, Depth V2 사용 시 ±20~30%. 항상 실제보다 가깝게 추정(보수적)합니다.
