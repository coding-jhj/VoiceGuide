| 항목 | 내용 |
|------|------|
| TTS 목소리 겹침 완전 해결 | STT 시작 시 TTS 즉시 중단, stale 재생 방지, MediaPlayer 단일 인스턴스 |

## 3. 현재 구현 완료된 기능 전체

### 핵심 파이프라인

- YOLO11m 장애물 탐지 (COCO 80클래스 + 계단 파인튜닝, mAP50 0.992)
- Depth Anything V2 상대 거리 추정 (GPU 실시간)
- 깊이 맵 기반 계단·낙차·턱 감지
- 경고 계층 분리 — critical(즉각 음성) / beep(비프음) / silent(무음)
- 한국어 조사 자동화 + 긴박도별 문장 생성

### Android 앱

- 1초마다 자동 캡처 → 온디바이스 ONNX 추론 (서버 없이 동작)
- STT 음성 명령 22가지 모드
- 카메라 방향 자동 감지 (가속도 센서)
- 개인 네비게이팅 ⭐ (장소 저장/찾기)
- 보호자 SOS ⭐, 낙상 감지 ⭐, 버스 번호 인식 ⭐
- 지폐 인식, 식사 도우미 모드, GPS 하차 알림, 약 복용 알림

## 4. 데모 시나리오 (실기기)

> 기기: Samsung Galaxy Z Fold 4 / 서버: RTX 5060 PC

1. 앱 시작 → "음성 안내를 시작할까요?" 자동 음성 → "네"로 시작
2. 장애물 앞에서 **1초마다 자동 분석** → "바로 앞에 의자가 있어요. 기다리세요."
3. "가방 찾아줘" → "가방은 왼쪽 앞 가까이 거리에 있어요."
4. "여기 저장해줘 편의점" → "편의점을 저장했어요."
5. 차량 접근 → `critical` 모드로 즉각 음성 끊고 경고

1. **경고 피로 해결** — critical/beep/silent 3단계로 불필요한 음성 최소화
