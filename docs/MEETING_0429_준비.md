## 실험/연구 노트 (강사님 필수 기록 항목)

### [A] YOLO11m COCO80 탐지 — 운영 중

| 항목 | 내용 |
|------|------|
| **베이스 모델** | YOLO11m (COCO80 사전학습) |
| **클래스 수** | 80 (계단 클래스 오탐률 문제로 제외, Depth 맵으로 대체) |
| **이미지 크기** | 640×640 |
| **클래스별 conf 임계값** | 차량 0.35–0.38 (야외 빠른 감지), 칼·가위 0.42–0.45 |
| **오인식 빈번 클래스** | 계단·노트북·냉장고·인형 → 제외 처리 |
| **IoU 중복 제거** | 같은 클래스 bbox IoU 0.3 이상 시 confidence 낮은 것 제거 |
| **VotingBuffer** | 10프레임 중 60% 이상 탐지된 물체만 경고 (오탐 차단) |
| **검증 현황** | 서버 9개 엔드포인트 자동 테스트 완료 |
| **미완료** | 실환경 클래스별 인식률 측정 (수치 추가 필요) |

### [B] Depth Anything V2 거리 추정 — 운영 중

| 항목 | 내용 |
|------|------|
| **모델** | Depth Anything V2 Small (vits, 24.8M params) |
| **입력** | 640px 리사이즈 JPEG (앱에서 75% 압축 후 전송) |
| **출력** | 역깊이 맵 → 5단계 거리 (0.5m / 1m / 2m / 3.5m / 5m+) |
| **보수 추정** | bbox 내 픽셀 하위 30% 사용 (가장 가까운 점 기준) |
| **계단/낙차 감지** | 바닥 12구역 깊이 분산 분석 |
| **Fallback** | Depth 모델 없을 시 → bbox 면적 비율로 거리 추정 |
| **검증 현황** | `/health` 엔드포인트로 로드 상태 확인 완료 |
| **미완료** | 실환경 거리 임계값 정량 측정 (4/28 기준 진행 중) |
