# VoiceGuide MVP 정의서 및 체크리스트

## MVP 목표

시각장애인이 스마트폰 카메라로 주변을 찍으면, AI가 장애물의 **위치·방향·거리·행동 안내**를 음성으로 실시간 안내한다.

---

## 데모 환경

| 항목 | 내용 |
|------|------|
| **실기기** | Samsung Galaxy Z Fold 4 (SM-F936N) |
| **에뮬레이터** | 사용 안 함 (실기기 데모) |
| **서버** | PC (Intel Core Ultra 7 265K) |
| **연결 방식** | 같은 Wi-Fi 내 로컬 IP (`http://192.168.x.x:8000`) |
| **데모 시나리오** | 폰 들고 장애물 앞 이동 → 자동 분석 → 음성 안내 |

- **플랫폼**: PC CPU (Intel Core Ultra 7 265K)

- **신뢰도 임계값**: 0.45
- **추론 방식**: Android 앱 → 이미지 전송 → FastAPI 서버 추론 → 결과 반환

## 기능 체크리스트

### 핵심 파이프라인

- [x] YOLO11m 장애물 탐지 (COCO 80클래스)
- [x] 방향 판단 → 이미지 9구역, 시계 방향(8시~4시) → 자연어 변환
- [x] 거리 추정 → bbox 면적 비율 기반, 5단계
- [x] 거리 수치 출력 → 1m 미만은 센티미터, 1m 이상은 미터
- [x] 위험도 계산 → 방향·거리 가중치 곱, 내림차순 정렬
- [x] 한국어 문장 생성 → 긴박도별 구조
- [x] 이/가 조사 자동화 → 받침 유무 판단
- [x] TTS 음성 출력 → Android 기본 TTS (한국어, 미디어 볼륨 스트림)

### 서버 / API

- [x] FastAPI `/detect` 엔드포인트 (이미지 POST → 문장 반환)
- [x] `camera_orientation` 파라미터 지원 (front/back/left/right)
- [x] 공간 스냅샷 DB (SQLite) → 장소별 변화 감지
- [x] 서버 시작 시 YOLO 워밍업 (첫 요청 지연 방지)
- [x] Gradio 데모 UI (localhost:7860)

### Android 앱

- [x] CameraX 라이브 프리뷰 (실시간 카메라 화면)
- [x] 2초마다 자동 캡처 → 서버 전송 → 결과 수신
- [x] 분석 시작 즉시 첫 캡처 (대기 없음)
- [x] Android 기본 TTS 한국어 음성 출력
- [x] 같은 문장 반복 안내 방지
- [x] TTS 말하는 중 텍스트 교체 방지
- [x] 로컬 IP 직접 연결 지원 (ngrok 없이 빠른 응답)

### 미완성 (다음 단계)

- [ ] Depth Anything V2 거리 추정 활성화
- [ ] STT 키워드 인식
- [ ] 카메라 방향 자동 감지 (나침반 + 가속도계)

## 방향 구역

| 이미지 x 비율 | 시계 방향 | 자연어 표현 | 위험도 가중치 |
|-------------|---------|-----------|------------|
| 0.00 ~ 0.11 | 8시 | 왼쪽 | 0.3 |
| 0.11 ~ 0.33 | 9~10시 | 왼쪽 / 왼쪽 앞 | 0.5 ~ 0.7 |
| 0.33 ~ 0.56 | 11~12시 | 왼쪽 앞 / 바로 앞 | 0.9 ~ 1.0 |
| 0.56 ~ 0.78 | 1~2시 | 오른쪽 앞 | 0.9 ~ 0.7 |
| 0.78 ~ 1.00 | 3~4시 | 오른쪽 | 0.5 ~ 0.3 |

---

## 거리 구역

| bbox 면적 비율 | 대략 거리 | 레이블 | 위험도 가중치 |
|-------------|---------|--------|------------|
| > 0.25 | ~0.5m | 매우 가까이 | 1.0 |
| > 0.12 | ~1.0m | 가까이 | 0.8 |
| > 0.04 | ~2.0m | 보통 | 0.5 |
| > 0.01 | ~4.0m | 멀리 | 0.2 |
| ≤ 0.01 | ~4.0m+ | 매우 멀리 | 0.1 |

## 실행 방법

```bash
# 서버 실행
cd C:\VoiceGuide\VoiceGuide
uvicorn src.api.main:app --host 0.0.0.0 --port 8000

# 로컬 IP 확인 (같은 Wi-Fi 권장)
ipconfig

# Android 앱 URL 입력
http://192.168.x.x:8000   # 로컬 IP (빠름, 권장)
https://xxxx.ngrok-free.app  # ngrok (다른 네트워크)
```
